🚀 Step 1: Create OCI Instance

  1. Log in to your Oracle Cloud Console
  2. Navigate to Compute → Instances
  3. Click Create Instance
  4. Configure your instance:
    • Name: Choose a descriptive name
    • Image: Ubuntu 22.04 LTS (Minimal)
    • Shape: VM.Standard.A1.Flex (2 OCPU, 12GB RAM) or VM.Standard.E2.1.Micro (Free tier)
    • Networking: Default VCN with public subnet
    • SSH Keys: Paste the public key above
  5. Click Create and wait for the instance to be running

⚙️ Step 2: Configure Security List

  1. In your instance details, click on the Subnet name
  2. Click on Default Security List
  3. Add these Ingress Rules:
    • HTTP: Port 80, Source: 0.0.0.0/0
    • HTTPS: Port 443, Source: 0.0.0.0/0
    • K8s API: Port 6443, Source: 0.0.0.0/0
    • SSH: Port 22, Source: 0.0.0.0/0 (if not already present)
